## 🚀 StudyFlare – Real-Time Study Partner Matching App

StudyFlare is a location-aware, real-time mobile application designed to simplify and accelerate the process of finding compatible study partners. By combining interest-based filtering, group size preferences, and live location data, the platform enables users to connect with nearby learners instantly through an intuitive “Tap to Study” experience.

---

## 💡 Problem Statement

Traditional methods of finding study partners are inefficient, time-consuming, and limited to existing social circles. There is no seamless way to instantly discover and connect with nearby students who share similar academic goals.

---

## ✅ Solution Overview

StudyFlare introduces a **real-time matching system** powered by:

* 📍 Location-based discovery
* 🎯 Interest-based filtering
* 👥 Group size selection

This enables users to instantly form study sessions with relevant peers using a single tap.

## ⚙️ System Workflow

1. User logs in and sets preferences
2. User taps **“Tap to Study”**
3. System fetches nearby users via GPS
4. Applies filters (interests + group size)
5. Displays matches using radar UI
6. User connects and starts session
7. Feedback collected to improve system
